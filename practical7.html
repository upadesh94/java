Practical 7: Interfaces and Packages

Implementation of Packages and Interfaces in Java

Problem Statement

Implementation of Java packages and interfaces demonstrating temperature conversion and interface implementation.

Part 1: Temperature Converter Package

This program demonstrates:

  • Package creation and usage
  • Temperature conversion methods
  • Class implementation in packages
  • Method overloading for different conversions

// TemperatureConverter.java
package pack1;

public class TemperatureConverter {
    public double Celsius_to_Fahrenhite(double c) {
        double far = c * 1.8 + 32;
        return far;
    }
    
    public double Fahrenhite_to_Celsius(double f) {
        double cel = (f - 32) * (5/9);
        return cel;
    }
}

// Main.java
import pack1.TemperatureConverter;

class Main {
    public static void main(String[] args) {
        TemperatureConverter t = new TemperatureConverter();
        System.out.println("Temperature in Fahrenhite = " + t.Celsius_to_Fahrenhite(32));
        System.out.println("Temperature in celsius = " + t.Fahrenhite_to_Celsius(32));
    }
}

Part 2: Interface Implementation

This program demonstrates:

  • Interface creation and implementation
  • Method declaration in interfaces
  • Class implementing interface
  • Package organization

// Interface.java
package packageEx;

public interface Interface {
    void displayMessage(String message);
    int addNumbers(int a, int b);
}

// InterfaceImpl.java
package packageEx;

public class InterfaceImpl implements Interface {
    public void displayMessage(String message) {
        System.out.println(message);
    }
    
    public int addNumbers(int a, int b) {
        return a + b;
    }
}

// PacEx.java
import packageEx.InterfaceImpl;

public class PacEx {
    public static void main(String[] args) {
        InterfaceImpl A = new InterfaceImpl();
        A.displayMessage("Hello!! Himanshu");
        int result = A.addNumbers(50, 50);
        System.out.println("Your Result is " + result + "% !!");
    }
}

How it works

  1. Temperature Converter Package:
    • Package 'pack1' contains TemperatureConverter class
    • Methods for Celsius to Fahrenheit conversion
    • Methods for Fahrenheit to Celsius conversion
    • Main class imports and uses the package
  2. Interface Implementation:
    • Interface defines method contracts
    • InterfaceImpl class implements the interface
    • Methods are implemented in the class
    • Main class demonstrates interface usage
