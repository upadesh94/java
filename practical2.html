Practical 2: Student Marks Management

Store and retrieve student marks using arrays

Problem Statement

Write a program to store five students' marks along with roll number in an array. Display marks of a particular roll number.

Solution

This program demonstrates:

  • Array usage for storing multiple student records
  • User input handling using Scanner class
  • Searching through arrays
  • Basic error handling for non-existent roll numbers

//Ass 2
import java.util.*;
class studentInfo{
    public static void main (String arr[])
    {
        Scanner scanner = new Scanner(System.in);
        int[] marks = new int [5];
        int[] rollno = new int [5];
        int i;
        int flag = 0;

        for(i = 0 ; i < marks.length ;i++){
            System.out.println("Enter the marks of student " + (i+1));
            marks[i] = scanner.nextInt();
            System.out.println("Enter the roll no of student " + (i+1));
            rollno[i] = scanner.nextInt();
        }
        System.out.println("Enter the roll no of student to search");
        int roll = scanner.nextInt();
        for(i = 0 ; i < rollno.length ; i++){
            if(roll == rollno[i]){
                System.out.println("Roll no " + roll + " has marks " + marks[i]);
                flag = 1;
                break;
            }
        }
        if (flag == 0){
            System.out.println("Roll no " + roll + " not found");
            }
    }
}

How it works

  1. The program creates two arrays:
    • marks[] to store student marks
    • rollno[] to store corresponding roll numbers
  2. It uses a loop to input data for 5 students:
    • Marks for each student
    • Roll number for each student
  3. After collecting all data, it asks for a roll number to search
  4. The program then:
    • Searches through the rollno array for a match
    • If found, displays the corresponding marks
    • If not found, displays an error message
  5. A flag variable is used to track whether the roll number was found
