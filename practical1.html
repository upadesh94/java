Practical 1: String Operations

Implementation of String Operations, Buffer String Operations, and Palindrome String Check

Problem Statement

Implementation of various string operations in Java including string manipulation, buffer string operations, and palindrome string checking.

1. Basic String Operations

This program demonstrates various operations that can be performed on String objects in Java, including:

  • Finding string length
  • Converting to uppercase and lowercase
  • Finding index of a substring
  • Extracting substrings
  • Checking string containment
  • String comparison

// Assignment 1: Operations on String
class StringOperation {
    public static void main(String args[]) {
        String s1 = "Welcome to MIT";
        
        // Length
        int l = s1.length();
        System.out.println("Length of the string is: " + l);
        
        // Uppercase
        String s2 = s1.toUpperCase();
        System.out.println("String in uppercase: " + s2);
        
        // Lowercase
        s2 = s1.toLowerCase();
        System.out.println("String in lowercase: " + s2);

        // Index of "MIT"
        int i = s1.indexOf("MIT");
        System.out.println("Index of 'MIT' in the string is: " + i);
        
        // Substring
        s2 = s1.substring(0, 5);
        System.out.println("Substring (0 to 5): " + s2);

        // Contains
        if (s1.contains("MIT")) {
            System.out.println("String contains 'MIT'");
        } else {
            System.out.println("String does not contain 'MIT'");
        }

        // Equals
        if (s1.equals("Welcome to MIT")) {
            System.out.println("String is equal to 'Welcome to MIT'");
        } else {
            System.out.println("String is not equal to 'Welcome to MIT'");
        }

        // Equals Ignore Case
        if (s1.equalsIgnoreCase("To")) {
            System.out.println("String is equal to 'To'");
        } else {
            System.out.println("String is not equal to 'To'");
        }
    }
}

2. StringBuffer Operations

This program demonstrates operations that can be performed using StringBuffer, which is mutable unlike String:

  • Appending text
  • Inserting text at specific positions
  • Reversing the string
  • Deleting portions of the string

// Assignment 2: Operations on StringBuffer
class BufferStringOperation {
    public static void main(String[] args) {
        StringBuffer s1 = new StringBuffer("Welcome to MIT");

        // Append
        s1.append(" college");
        System.out.println("\nStringBuffer after append operation: " + s1);

        // Insert
        s1.insert(11, " My");
        System.out.println("After insert operation: " + s1);

        // Reverse
        s1.reverse();
        System.out.println("After reverse operation: " + s1);

        // Delete
        s1.delete(0, 5);
        System.out.println("After delete operation: " + s1);
    }
}

3. Palindrome Check

This program demonstrates how to check if a given string is a palindrome (reads the same forwards and backwards):

  • Takes user input for a string
  • Uses StringBuffer to reverse the string
  • Compares the original and reversed strings
  • Determines if the string is a palindrome

// Assignment 3: Palindrome Check
import java.util.*;

class Palindrome {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("Enter any string: ");
        String name = sc.nextLine();

        StringBuffer s1 = new StringBuffer(name);
        s1.reverse();

        if (name.equals(s1.toString())) {
            System.out.println("Given string is a palindrome");
        } else {
            System.out.println("Given string is not a palindrome");
        }
    }
}
