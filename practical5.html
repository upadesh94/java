Practical 5: Shape Calculator

Implementation of Area and Perimeter Calculator for Different Shapes Using Inheritance

Problem Statement

Implementation of Area and Perimeter Calculator for Different Shapes Using Inheritance in Java.

Solution

This program demonstrates:

  • Object-Oriented Programming concepts
  • Inheritance in Java
  • Method overriding
  • Abstract class implementation
  • Mathematical calculations

class Shape {
    double len;
    double hig;
    double radius;
    double area;
    double base;

    void getArea() {}
    void getPerimeter() {}
}

class Circle extends Shape {
    void setRadius(double radius) {
        this.radius = radius;
    }

    @Override
    void getArea() {
        area = Math.PI * radius * radius;
        System.out.println("Area of Circle is " + area);
    }

    @Override
    void getPerimeter() {
        double perimeter = 2 * Math.PI * radius;
        System.out.println("Perimeter of Circle is " + perimeter);
    }
}

class Rectangle extends Shape {
    void setDimensions(double len, double hig) {
        this.len = len;
        this.hig = hig;
    }

    @Override
    void getArea() {
        area = len * hig;
        System.out.println("Area of Rectangle is " + area);
    }

    @Override
    void getPerimeter() {
        double perimeter = 2 * (len + hig);
        System.out.println("Perimeter of Rectangle is " + perimeter);
    }
}

class Triangle extends Shape {
    void setDimensions(double base, double hig) {
        this.base = base;
        this.hig = hig;
    }

    @Override
    void getArea() {
        area = 0.5 * base * hig;
        System.out.println("Area of Triangle is " + area);
    }

    @Override
    void getPerimeter() {
        System.out.println("Perimeter of Triangle cannot be calculated without all 3 sides.");
    }
}

public class cal_area {
    public static void main(String[] args) {
        Circle c = new Circle();
        c.setRadius(4.0);
        c.getArea();
        c.getPerimeter();

        System.out.println(); // spacer

        Triangle t = new Triangle();
        t.setDimensions(6.0, 4.0);
        t.getArea();
        t.getPerimeter();

        System.out.println(); // spacer

        Rectangle r = new Rectangle();
        r.setDimensions(4.0, 3.0);
        r.getArea();
        r.getPerimeter();
    }
}

How it works

  1. Class Hierarchy:
    • Shape: Base class with common attributes and empty methods
    • Circle: Extends Shape with radius-based calculations
    • Rectangle: Extends Shape with length and height calculations
    • Triangle: Extends Shape with base and height calculations
  2. Features:
    • Area calculation for each shape
    • Perimeter calculation (where applicable)
    • Method overriding for specific calculations
    • Input validation and error handling
  3. Implementation Details:
    • Uses Math.PI for circle calculations
    • Different formulas for each shape
    • Triangle perimeter calculation limitation noted
    • Clean output formatting
