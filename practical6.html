Practical 6: Sports Score Calculator

Implementation of Score Calculator for Different Sports Using Method Overloading

Problem Statement

Implementation of Score Calculator for Different Sports Using Method Overloading in Java.

Solution

This program demonstrates:

  • Method Overloading in Java
  • Polymorphism
  • Different parameter types and counts
  • Sports score calculations
  • Clean code organization

public class sportScore {

    // Cricket score: runs/wickets
    void calculateScore(int runs, int wickets) {
        System.out.println("Cricket Score: " + runs + "/" + wickets);
    }

    // Football score: goals
    void calculateScore(int goals) {
        System.out.println("Football Score: " + goals + " goals");
    }

    // Basketball score: calculated from point types
    void calculateScore(int twoPointers, int threePointers, int freeThrows) {
        int totalScore = (2 * twoPointers) + (3 * threePointers) + (1 * freeThrows);
        System.out.println("Basketball Score: " + totalScore + " points");
    }

    public static void main(String[] args) {
        sportScore score = new sportScore();

        score.calculateScore(250, 5);         // Cricket
        score.calculateScore(3);              // Football
        score.calculateScore(9, 5, 4);        // Basketball
    }
}

How it works

  1. Method Overloading:
    • Same method name calculateScore with different parameters
    • Different parameter counts for each sport
    • Compiler determines which method to call based on arguments
  2. Sports Implementation:
    • Cricket: Takes runs and wickets as parameters
    • Football: Takes only goals as parameter
    • Basketball: Takes different types of shots and calculates total points
  3. Features:
    • Clean and intuitive method names
    • Sport-specific score calculations
    • Formatted output for each sport
    • Easy to extend for more sports

Sample Output

Cricket Score: 250/5
Football Score: 3 goals
Basketball Score: 31 points
