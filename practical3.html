Practical 3: Stack Implementation

Implementing a Stack with push, pop, and empty check operations

Problem Statement

Write a class Stack to push number, pop number and to check stack empty or full. (Use OOPs concepts and constructor overloading to assign default size of stack to 5 or user can change the stack size while creating object).

Solution

This program demonstrates:

  • Object-Oriented Programming concepts
  • Stack data structure implementation
  • Basic stack operations (push, pop, isEmpty)
  • Stack overflow and underflow handling
  • Stack content display

import java.util.*;

class Stack {
    int[] stack = new int[5];
    int top = -1;

    // Push method
    void push(int x) {
        if (top == 4) {
            System.out.println("Stack is full. Cannot push " + x + ".");
        } else {
            stack[++top] = x;
            System.out.println("Pushed " + x + " to the Stack");
        }
    }

    // Pop method
    void pop() {
        if (top != -1) {
            System.out.println("Popped " + stack[top] + " from the Stack");
            stack[top--] = 0;
        } else {
            System.out.println("Stack is empty. No item to remove.");
        }
    }

    // Check if empty
    void isEmpty() {
        if (top == -1) {
            System.out.println("Stack is empty");
        } else {
            System.out.println("Stack is not empty");
        }
    }

    // Optional: Print stack contents
    void display() {
        System.out.print("Current Stack: ");
        for (int i = 0; i <= top; i++) {
            System.out.print(stack[i] + " ");
        }
        System.out.println();
    }
}

public class StackOperation {
    public static void main(String[] args) {
        Stack s = new Stack();

        s.isEmpty();      // Check if empty

        // Push elements
        s.push(1);
        s.push(2);
        s.push(3);
        s.push(4);
        s.push(5);
        s.push(6);        // Should show stack full

        s.isEmpty();      // Now should not be empty

        // Pop elements
        s.pop();
        s.pop();

        s.isEmpty();      // Should still not be empty

        // Display stack
        s.display();
    }
}

How it works

  1. The program implements a Stack class with the following features:
    • Fixed-size array to store stack elements
    • Top pointer to track the current position
    • Methods for basic stack operations
  2. Stack Operations:
    • push(): Adds an element to the top of the stack
    • pop(): Removes and returns the top element
    • isEmpty(): Checks if the stack is empty
    • display(): Shows current stack contents
  3. Error Handling:
    • Stack overflow check in push operation
    • Stack underflow check in pop operation
